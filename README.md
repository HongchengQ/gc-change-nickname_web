
# gc-change-nickname_web
使用web轻松更改gc数据库中的玩家昵称 只需要填写uid和要更改的昵称

1.安装Flask、pymongo和waitress这些依赖库
你可以使用以下命令来安装它们
```
pip3 install flask pymongo waitress
```

2.双击运行.run.bat文件

3.浏览器访问你的ip:21472

(127.0.0.1:21472)
